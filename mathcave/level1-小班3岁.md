# Level 1 — 小班（3岁）数学题库

> 参考：《3-6岁儿童学习与发展指南》、人教版幼儿园教材
> 大脑特点：直观感知、短时记忆、只能理解眼前实物，无抽象数字概念

---

## 一、数感 — 点数与认数

### 1.1 手口一致点数（1-5）
```
题：数一数，图中有几个🍎？
🍎 🍎 🍎
A. 2  B. 3  C. 4  D. 5
答案：B

题：数一数，下面有几只🐱？
🐱 🐱 🐱 🐱
A. 3  B. 4  C. 5  D. 6
答案：B
```

### 1.2 按数取物
```
题：请拿出 3 个积木
（展示 1-5 个物品，要求选出指定数量）
```

### 1.3 区分多少
```
题：哪边多？
左边：🌟🌟🌟
右边：🌟🌟🌟🌟🌟
A. 左边多  B. 右边多  C. 一样多
答案：B
```

## 二、图形与空间

### 2.1 认识基础图形
```
题：下面哪个是圆形？
A. □  B. ○  C. △
答案：B

题：哪个是三角形？
A. ○  B. □  C. △
答案：C
```

### 2.2 空间方位
```
题：小猫在桌子的____面？（上/下）
题：球在盒子的____面？（里/外）
```

---

## 三、比较与分类

### 3.1 大小、长短、高矮
```
题：哪个苹果最大？
🟢（小） 🔴（中） 🟡（大）
A. 绿的  B. 红的  C. 黄的
答案：C
```

### 3.2 简单分类
```
题：把红色的放在一起
🔴 🔵 🔴 🟡 🔴 🔵
选出所有红色的（按颜色分类）
```

---

## 四、规律推理

### 4.1 AB 规律
```
题：找规律：🔴 🔵 🔴 🔵 🔴 ？
接下来是：
A. 🔴  B. 🔵
答案：B

题：找规律：⭐ 🌙 ⭐ 🌙 ？
接下来是：
A. ⭐  B. 🌙
答案：A
```
